# audiobook
Read any audio book using only 12 lines of Python.

### Tutorial
[Video Tutoraial](https://youtu.be/kyZ_5cvrXJI)

### Installation
To read the PDF file using Python
` pip install pypdf2
`
To Read the text (Text to speech)
` pip install pyaudio
`
` pip install pyttsx3
`

#### Note:
This code can read all the printable text from a PDF file / Book
